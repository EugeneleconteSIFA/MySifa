Mise à jour — v1.5.0
Nouveautés
  • Nouveau module Impression : les étiquettes partent directement du saas vers les imprimantes Zebra, Brother et thermiques de l'usine, même quand vous êtes en 5G sur le téléphone.
  • Réception matière — le bouton Imprimer envoie désormais les étiquettes d'identification bobine sur l'imprimante configurée pour votre poste. Fallback navigateur disponible.
  • Chaque utilisateur peut retenir son imprimante par défaut par usage. Le picker s'affiche uniquement au premier envoi ou en cas de changement de poste.
  • Nouvel onglet Paramètres > Imprimantes (super admin) : gestion des imprimantes, des templates ZPL/EPL et des agents locaux. Bouton "Test d'impression" par imprimante.
Technique
  • Nouvel agent Python autonome à déployer sur un Raspberry Pi de l'usine (tools/print_agent/). Poll HTTPS sortant, zéro port ouvert côté LAN.
  • Migration DB v171 — cinq tables : print_agents, imprimantes, imprimante_templates, print_jobs, user_printer_defaults.
Dans l'optique d'améliorer constamment l'outil, vos retours sont les bienvenus.
